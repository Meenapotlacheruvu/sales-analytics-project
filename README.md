# Sales Analytics Project

## Objective
Analyze retail sales data to identify revenue trends and regional performance.

## Tools
Python, Pandas, Matplotlib

## Key Insights
- West region generated the highest revenue.
- Sales growth increased significantly after 2016.
- Technology category drove the most sales.

## Project Files
- sales_analysis.ipynb : analysis notebook
- sales_growth.png : yearly sales trend
- sales_by_region.png : regional comparison
